Qt::QAbstractObject

DESCRIPTION

This module is the foundation class of PerlQt4onGtk. As an abstract class, this class is only for subclassing.

SYNOPSIS

    package MyObject;
    use parent 'Qt::QAbstractObject';
    
    sub new {
        my $class = shift;
        my $self = $class->SUPER::new(__PACKAGE__);
        return $self;
    }

METHODS

new($child_class_name)

When subclassing QAbstractObject, specify the subclass name as $child_class_name. It is a good idea to use __PACKAGE__. The $child_class_name will added to the meta object list of this object in order to identify the type of this widget.

The reason why we not using the default perl ref is that ref cannot recognize the class tree. And with the new meta mechanism, it is easy to find out what one object is and also what the base class of one object is.

is($class_name)

returns true if this widget is a class $class_name

example:

    my $hello = QPushButton("Hello world!");
    print $hello->is('QPushButton'),"\n"; #return 1
    print $hello->is('QAbstractWidget'),"\n"; #return 1
    print $hello->is('QLineEdit'),"\n"; #return 0

add_meta($class_name)

add the class name $class_name to the meta name list of this widget.

example:

    my $hello = QPushButton("Hello world!");
    print $hello->is('foo'),"\n"; #return 0
    $hello->add_meta('foo');
    print $hello->is('foo'),"\n"; #return 1

WARNING This method should be carefully called. Some internal mechanism, like the layout system, uses the meta class of one object.

example:

    my $hello = QPushButton("Hello world!");
    $hello->add_meta('QLineEdit'); #very bad

set_properity($properity_name,$properity_value)

set a properity valued $properity_value to properity name $properity_name.

properity($properity_name)

returns the properity valued of properity name $properity_name.

register_signal_map($signal_name_map)

Add signals to the signal list of the widget. $signal_name_map can be a list reference or hash reference.

example:

    sub slot {
        my ($p_connect,$p_emit) = @_;
        
        if (defined $p_connect) {
            print "\$p_connect = $p_connect\n";
        }
        if (defined $p_emit) {
            print "\$p_emit = $p_emit\n";
        }    
    }
    
    my $w = QObject();
    $w->register_signal_map(['signal1','signal2']);
    $w->{signal1}->connect(\&slot,'connect signal1');
    $w->{signal2}->connect(\&slot,'connect signal2');
    $w->{signal1}->emit('emit signal1');
